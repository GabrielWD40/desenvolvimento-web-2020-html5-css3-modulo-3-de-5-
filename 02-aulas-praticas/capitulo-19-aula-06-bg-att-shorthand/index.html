Coisas interessantes dessa aula

Nessa aula nós aprendemos a como deixar uma imagem de fundo fixa ou rolável! Isso é importante porque as vezes nós impedimos que as imagens quebrem se o conteúdo for muito grande. Temos duas opções de background-atachament:

  • background-atchament: fixed → deixa a imagem fixa e impede que a imagem quebre.

  • background-atchament: scroll → A imagem é rolável, não fica fixa.

Olha só como a imagem não quebra se o conteúdo for enorme.

Nós podemos usar um SHORTHAND para usar todas as propriedade de uma só vez. Usamos a tag "background", se liga na ordem:

  1. atributo color → cor de fundo
  2. atributo imagem → caminho da imagem de fundo
  3. atributo position → posição das matrizes 3x3
  4. atributo repeat → se vai repetir ou não e em quais direções (x ou y)
  5. atributo size → cover ou contain refere-se a como a imagem vai comporta-se na exibição
  6. atributo attchament → scroll ou fixed, se a imagem vai ser estática ou vai rolar junto com o site.

background: black url(caminho) top center no-repear [cover] fixed.

Por alguma razão, o atributo cover não está funcionando mesmo com a sintaxe certa. Aí você tem que usar o background-size: cover; automaticamente
