Nessa aula nós vamos brincar com o background repeat.

  • background-size: define o tamanho da imagem!

  • background-repeat: define o ciclo de repetição da imagem!
    Temos as seguintes propriedades:
    1. background-repeat: no-repeat; → não deixa a imagem se repetir
    2. background-repeat: repeat; → A imagem se repete no tamanho da tela.
    3. background-repeat: repeat-x; → A imagem só vai repetir no eixo x (DEITADO)
    4. background-repeat: repeat-y; → A imagem só vai se repetir na vertical (DE PÉ)

  • background-size: define o tamanho da imagem

Por padrão, as imagens da propriedade background-image vem com o background repeat. Além disso, o ponto de partida padrão é o canto superior esquerdo.
