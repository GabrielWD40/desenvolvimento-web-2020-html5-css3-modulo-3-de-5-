EFEITO ZEBRADO EM TÍTULOS!

Nessa aula, nós vamos aprender a usar o efeito zebrado dentro de tabelas. Ele é bem útil quando se tem muito dados na vertical e ajuda na leitura deles.

para usar o efeito zebrado vamos precisar usar uma pseudo classe chamdada :nth-child(xn),
onde x representa a sequencia
o N você coloca junto para informar que é em mais de uma linha. Se vc colocar só em uma linha ele vai aplicar só para o número da TR em questão. Ex: Se você colocar o número 5, ele vai aplicar o efeito bildado só para a linha 5.

MAAAAAAS....O professor deu uma dica muito melhor!

A dica dada por ele é usar as estilizações pares linhas que são pares e ímpares, isso facilita o trabalho!

odd → ímpar

even → par

Exemplificando os ímpares:
tbody > tr: nth-child(odd){
background-color: red;
color: white;
}
Nesse primeiro exemplo, o estilo com fundo vermelho e letra branca será aplicado somente a linhas ímpares


Exemplificando os ímpares:
tbody > th: nth-child(even){
background-color: green;
color: white;
}
Nesse segundo exemplo, o estilo de fundo verde com letra branca serão aplicadas somente as linhas pares
