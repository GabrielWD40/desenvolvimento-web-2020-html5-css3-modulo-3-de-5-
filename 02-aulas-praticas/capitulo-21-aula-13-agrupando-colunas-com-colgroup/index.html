Nessa aula nós falaremos sobre a tag COLGROUP.
Essa tag é útil para estilizarmos uma coluna por inteiro sem ficar inserindo classes / id linha a linha.
Isso torna o nosso código mais legível e mais fácil de manusear.

Exemplo dessa nova tag: 

        

            <colgroup>
                <col class="nome-da-classe-da-coluna-1"
                <col class="nome-da-classe-da-coluna-2"
                <col class="nome-da-classe-da-coluna-3"
                <col class="nome-da-classe-da-coluna-4 span 2"
                    no exemplo acima, a coluna 4 e 5 terão o mesmo efeito
            </colgroup>

É importante lembrar que o número de colunas deve ser o mesmo que as estilizações no colgroup. Se não for, você precisa usar a propriedade span para que ele aplique o mesmo estilo a mais de uma coluna, ok?
