Nessa aula nós vamos aprender a como centralizar uma div


LEMBRETE: Para centralizar verticalmente é necessáio ter um "container" e um "conteúdo."

O container deve ter o posicionamento relativo, enquanto o posicionamento do conteúdo deve ser absoluto.

Feito isso, nós conseguimos configurar a distância da esquerda e do topo.



Após isso, vamos colocar a caixa com posicionamento de top e left de 50%, o que irá jogar esse elemento para o centro do container.

Aí você precisará usar a propriedade
transform: translate(-50%, -50%);


Existem muitas propriedades dentro do transform. Mas essa aí serve para conseguirmos mover um elemento. Você precisa se lembrar que:

  • O primeiro valor vai alterar o lado esquerdo.
    Valores negativos puxam ele para a esquerda, positivos jogam ele para a direita.

  • O segundo valor vai alterar para cima ou para baixo. Se você usar valores positivos, o elemento será realocado para baixo. Se usar valores negativos, o elementos será alocado para cima.


Veja que o elemento sempre ficara no centro mesmo que o tamanho da caixa pai seja alterado.
